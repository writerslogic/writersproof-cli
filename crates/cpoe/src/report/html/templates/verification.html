The cryptographic evidence underlying this report can be independently verified by any party without reliance on the examiner or issuing organization:

  • Web verification: Upload the evidence file at writerslogic.com/verify. Verification executes entirely in the browser using WebAssembly; no document content is transmitted to the server.
  • Command-line verification: Install the open-source CPoE CLI (cpoe_cli) and execute cpoe verify <evidence-file>. Source code is available at the WritersLogic repository.
  • Manual verification: The checkpoint chain hashes can be recomputed from the raw evidence data using any SHA-256 implementation. Each checkpoint's HMAC-SHA256 binding to its predecessor can be independently verified. VDF proofs can be verified by re-executing the delay function for the claimed number of iterations.

Verification confirms: Ed25519 cryptographic signatures, checkpoint chain integrity (HMAC binding), VDF timing proof consistency, and document hash match. Verification does not re-evaluate behavioral metrics; it confirms the evidence was captured and sealed correctly.
